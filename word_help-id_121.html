东南亚国家瓦楞纸箱纸板生产标准要求

2018-05-08 00:22:00
中国纸箱网
语音播放

现在经济高速发展，中国和东南亚国家的经济往来越来越密切，小编整理了一下东南亚的瓦楞纸箱、瓦楞纸板生产标准及生产要求，希望对大家的出口贸易有帮助。

东南亚瓦楞纸箱主要用于包装运输，基本箱型可分为：

开槽型纸箱：为最常用的外包装纸箱，基本上由一片瓦楞纸板组成，通过钉合或粘合等方法将接缝封合制成纸箱，有顶部及底部折片构成箱底和箱盖。运输时可以折叠平放，使用时把箱盖封合。

套合型纸箱：由几顶箱组成的纸箱，其特点是箱体与箱盖时，箱盖和箱体是套接的。

折叠型纸箱：通常由一片瓦楞纸板组成，折叠而成箱的底粘合。

也可根据要求按订货合同所规定的其他型式制造。顶盖、底盖)分开，使用侧面和箱盖，不用钉。另外，为提高对内装物的保护能力，允许用隔挡、衬垫、底座、模切件等纸箱附件。

尺寸规定

纸箱的综合尺寸，是指其内尺寸长、宽、高之和。其中1类箱主要用于出口及贵重物品的运输包装；2类箱主要用于内销产品的运输包装；3类箱主要用于短途、低廉商品的运输包装。用纸箱的内尺寸长、宽、高表示纸箱的规格，并按下列顺序和代号给出尺寸，单位用mm。

根据内装物的重量和尺寸，优先选用以下底面积尺寸：400 x 600、400 x 300、400 x200、400 X 150、300 X 200、300 X 133、300 X 100、200 X 150、200 X 133。

也可根据被包装物品的特点选用其他的尺寸。箱子长、宽之比建议不大子2．5：1；高、宽之比不大于2：1并不小于0．15：1。

其他标准

■钉合瓦楞纸箱使用带有镀层的低碳钢扁丝，扁丝不应有锈斑、剥层、龟裂或其他使

■粘合瓦楞纸箱使用乙酸乙烯乳液或具有相同粘合效果的其他粘合剂。

■箱体方正，表面不允许有明显的损坏和污迹，切断口表面裂损宽度不超过8mm。

■箱面印刷图字清晰，深浅一致，位置；隹确，必须在箱盖内印刷纸箱制造厂名称或代号。

■纸箱接头钉合搭接舌边宽度35mm-50mm，金属钉应沿搭接部分中线钉合，采用斜钉(与纸箱立边成45°角)或横钉，箱钉应排列整齐、均匀，单排钉距不大于80mm，钉距均匀，头尾钉距底面压痕边线不大子20mm。钉合接缝应钉牢，不得有叠钉、翘钉、不转角等缺陷。

■纸箱接头粘合搭接舌边宽度不小于30mm，粘合剂使用乙酸乙烯乳液或具有相同粘合效果的其他粘合剂。粘合接缝的粘合剂涂布应均匀充分，以致面纸分离时接缝依然粘合不分，同时，也不应有多余的粘合剂溢出接缝。

■瓦楞纸箱的压痕线宽度，单瓦楞纸箱不大于?2mm，双瓦楞纸箱不大于17mm，折线居中，不得有破裂断线，箱壁不允许有多余的压痕线。

■瓦楞纸箱摇盖经开、合180往复5次以上，1、2类箱面层和里层都不得有裂缝，3类纸箱板外面层不得有裂缝，内面层裂缝长度总合不得大约70mm。

■瓦楞纸箱可根据供需双方的协定进行表面防潮处理。

■瓦楞纸箱的机械性能包括抗压强度、耐冲击强度、抗转载强度等，在每种具体产品用瓦楞纸箱的标准或技术要求中，明确规定或由供需双方商定。

■已制成的瓦楞纸箱，根据纸箱种类及需要，应进行纸板性能试验，如有需要可进行纸箱强度试验。

■测试纸箱内尺寸时，把纸箱撑开成型、相邻面夹角成90°，量取搭接舌边以外的、构成长宽两面间的距离(内尺寸)为箱长(L)、箱宽(W)，量取构成箱高的底、盖间的内距离为箱高(H)。

■瓦楞纸箱空箱承压强度试样制备时，用胶粘带、金属钉或黏合剂封合，并将采用的方法写在试验报告内，试样数量应不少于3只。

■钉合瓦楞纸箱使用16～18号带有镀层的金属扁丝，金属扁丝不许有龟裂或使用上的缺点。 

■瓦楞纸箱应成批交货。生产厂保证出厂的瓦楞纸箱符合本标准的规定，并出具合格证。

■需货方按本标：隹进行验收，当发现某项指标不符合标准要求时，可对不合格的指标进行复验。当内尺寸、厚度及耐破强度、边压强度、粘合强度、水分有一项不符合本标准时，或其他指标不符合标；隹的纸箱数多于10％时，则该批纸箱为不合格品。

■生产厂应根据用户要求对瓦楞纸箱进行包装。

■保证瓦楞纸箱在运输过程中不直接受雨、台风、曝晒和污染的影，向。装卸时不得从 

高处仍下，不得使用有损产品质量的工具。曰储存仓库应保持通风干燥，短期露天存放必须有良好的垫防护措施。
